We have to put in this file all the UI items, the standard and non-standard services which are not documented in this plug-in.

UI:
New documentation block
Documentation in hover and during autocompletion
Support for @TODO and @FIXME in comment and documentation blocks
Action to show invisible characters that will be generated by Acceleo
Support for warnings
Support for informations (green arrow on the left of a template that overrides another template)
New icons in the outline and the autocompletion popup to show the visibility of a template or a query
New actions in the outline to filter or modify the visible elements in the outline
Support for the @deprecated tag
